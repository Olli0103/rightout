# RightOut 0.5.0 parity and EU-process matrix

This matrix is a release contract. It compares capability classes, not marketing effectiveness or private database coverage.

| Capability | v0.5.0 evidence | Status |
| --- | --- | --- |
| Multi-subject private profiles | Up to 20 SecretRef profiles; opaque public IDs | complete |
| People-search discovery | 21 Brave-index lanes across 22 people-search entries | indirect-only |
| US email removal | BeenVerified, attested `US-CA`, prior discovery required | complete, narrow |
| US form removal | Intelius/PeopleConnect closed sandbox recipe | complete, narrow |
| EU controller email | Adsquare and emetriq, fixed GDPR kind/template, no prior listing required | complete, narrow |
| EU controller portals | Criteo and Zeotap official flows | human-only |
| EU advertising preferences | EDAA YourOnlineChoices and emetriq browser opt-out | human-only, not erasure |
| EU effect classification | process class, effect scope, erasure semantics, one-click level, official URL | complete |
| Removal confirmation | direct known-listing absence for supported US lanes only | scoped |
| EU outcome confirmation | controller response and human review | no automatic confirmation |
| Durable ledger and due work | encrypted cases, 30-day EU follow-up, recurring recheck tool | complete |
| Native approval | separate optional/non-replay-safe tool call and `allow-once` for each external effect | complete |
| Managed-service breadth | no dashboard, broad private inventory, family admin, human service, or effectiveness guarantee | not parity |

Hermes Unbroker remains broader in executable people-search forms. Incogni, Optery, DeleteMe, and similar services claim substantially broader inventories, dashboards, repeated submissions, proof/status workflows, and human escalation. RightOut remains a transparent, operator-run OpenClaw plugin with minimum workflow feature parity and explicit evidence gaps.

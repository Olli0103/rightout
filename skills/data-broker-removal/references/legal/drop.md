# California DROP Notes

California DROP source: `https://privacy.ca.gov/drop/`

Key operational facts verified 2026-07-11:

- DROP is for California residents.
- It lets eligible users tell data brokers to delete and not sell personal information.
- It requires residency verification through California's verification flow.
- DROP launches January 1, 2026; brokers begin processing requests August 1, 2026.
- Data brokers delete data every 45 days after processing begins.

OpenClaw rule: treat DROP as `human_only` until a subject's California residency and submission approval are explicit. Do not automate identity verification.
